.container

Este es un contenedor de ancho fijo. Se ajusta a diferentes tamaños de pantalla con puntos de interrupción.

.container-fluid

Este es un contenedor de ancho completo para todas las ventanas y tamaños de pantalla.

.container-sm

Este contenedor es fluido hasta el punto de interrupción pequeño (sm), luego se convierte en fijo.

.container-md

Este contenedor es fluido hasta el punto de interrupción mediado (md), luego se convierte en fijo.

.container-lg

Este contenedor es fluido hasta el punto de interrupción grande (lg), luego se convierte en fijo.

.container-xl

Este contenedor es fluido hasta el punto de interrupción extra grande (xl), luego se convierte en fijo.

.container-xxl

Este contenedor es fluido hasta el punto de interrupción extra-extra grande (xxl), luego se convierte en fijo.
